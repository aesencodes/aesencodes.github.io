Face Recognition Attendance System

Biometric solution eliminating attendance fraud with facial recognition technology

May 2025
Computer Science Thesis
800+ users

Project Overview

Developed as my bachelor's thesis at Institut Teknologi Sumatera, this system addresses attendance fraud limitations in traditional fingerprint systems with:

  • Python-based facial recognition for accurate biometric verification
  • Web-based administration portal for management
  • Real-time tracking with geolocation validation

Technical Implementation

Technology Stack

  • Python, OpenCV, Dlib
  • Flask RESTful API
  • CodeIgniter 3 (PHP)
  • MySQL with face encodings

Key Features

  • 98.7% recognition accuracy
  • Anti-spoofing liveness detection
  • Real-time reporting dashboard
  • Geofencing security

System Architecture

1

Face Enrollment

Staff register through web portal with multiple facial captures from different angles

2

Recognition Process

Python service processes camera feed and matches against database

3

Data Integration

Flask API records attendance events displayed in admin portal
