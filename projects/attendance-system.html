Face Recognition Attendance System

Biometric solution eliminating time fraud and streamlining university attendance tracking

Completed: May 2025
Project: Computer Science Thesis
Users: 800+ university staff

Project Overview

The Face Recognition Attendance System was developed as my bachelor's thesis to address persistent attendance fraud and limitations in the existing fingerprint-based system at Institut Teknologi Sumatera. This hybrid solution combines:

  • Python-based facial recognition for accurate biometric verification
  • Web-based administration portal for management and reporting
  • Real-time attendance tracking with geolocation validation

Technical Implementation

Technology Stack

  • Facial Recognition: Python, OpenCV, Dlib
  • Backend API: Flask RESTful API
  • Admin Interface: CodeIgniter 3 (PHP)
  • Database: MySQL with optimized face encodings

Key Features

  • 98.7% accurate face recognition in varied lighting
  • Anti-spoofing with liveness detection
  • Real-time attendance reporting dashboard
  • Geofencing to prevent remote clock-ins

System Architecture

1

Face Enrollment

Staff members register through the web portal with multiple facial captures from different angles

2

Recognition Process

Python service processes camera feed, extracts 128-dimension face encodings, and matches against database

3

Data Integration

Flask API records attendance events which are displayed in the CodeIgniter admin portal
